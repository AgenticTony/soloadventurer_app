## Flutter Test Runner

I'll run your Flutter tests with coverage and show you the results.

**Running:** `flutter test --coverage`

This will:
1. Run all unit and widget tests
2. Generate coverage report
3. Show test results with failures highlighted

After the tests complete, I'll:
- Show summary of passed/failed tests
- Display any failing tests with error messages
- Point to coverage report for detailed analysis
