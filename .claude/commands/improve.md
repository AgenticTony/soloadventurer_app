## Workflow Improver

I'll analyze and improve our development workflow.

**What I analyze:**
1. Current skill effectiveness
2. Repetitive patterns that could be automated
3. Bottlenecks in development flow
4. Quality issues that keep appearing
5. Opportunities for new skills

**Process:**
```
🔍 Analysis Phase
├── Review recent tasks
├── Identify friction points
├── Find patterns
└── Suggest improvements

💡 Implementation Phase
├── Create new skills
├── Update existing skills
├── Add verification loops
└── Document best practices

🔄 Validation Phase
├── Test new workflow
├── Measure improvement
└── Iterate if needed
```

**Example:**
```
"I notice you always run:
/test → /analyze → /gen → /test

Creating /full-check skill to automate this..."
```

**The workflow improves itself!** 🔄♾️
