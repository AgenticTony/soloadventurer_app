## Code Analysis & Fixes

I'll analyze your Flutter code and automatically fix what we can.

**Steps:**
1. Run `flutter analyze` to find issues
2. Run `dart fix --apply` for automatic fixes
3. Show remaining issues that need manual attention

**Focus areas:**
- Riverpod provider patterns
- Supabase integration
- Clean architecture violations
- Performance issues
- Type safety

After analysis, I'll:
- Show fixed issues count
- List remaining problems by severity
- Suggest specific fixes for each issue
